# Java秒杀系统方案设计

### 前言

秒杀系统相较于普通系统有很大的不同。在开始秒杀的瞬间，秒杀系统会面临两大挑战：(1)系统的并发量会非常的大；(2)并发量大的同时，网络的流量也会瞬间变大。

如果不针对秒杀功能进行优化，那么服务器的 load 瞬间会超出机器的承载能力，导致应用瘫痪甚至宕机。另外，秒杀系统在处理业务时要解决超卖、恶意刷单等问题。
